# Backup & Disaster Recovery Lab

> Recoverability proven with regular point-in-time restore drills.

## Overview

A resilience lab built on Proxmox Backup Server and Veeam, with scheduled, deduplicated backups across VMs and containers, automated restore testing, and tiered retention.

## Tech stack

- Veeam
- Proxmox Backup Server
- Restore Testing
- Retention

## What this lab demonstrates

- Scheduled, deduplicated backups across VMs and containers
- Automated restore testing
- Tiered retention policies
- Regular point-in-time restore drills
